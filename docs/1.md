# 项目

> 贡献者：[@zhongjiajie](https://github.com/zhongjiajie)

## 历史

Airflow于2014年10月由Airbnb的Maxime Beauchemin开始。它是第一次提交的开源，并在2015年6月宣布正式加入Airbnb Github。

该项目于2016年3月加入了Apache Software Foundation的孵化计划。
